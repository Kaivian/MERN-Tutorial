# MERN Tutorial  
MongoDB - Express.js - React - Node.js (Fullstack Web Application)

---

## Giới thiệu  
Dự án này được xây dựng nhằm hướng dẫn phát triển **ứng dụng web fullstack** với **MERN Stack**, bao gồm:  
- **MongoDB** – Cơ sở dữ liệu NoSQL  
- **Express.js** – Framework cho backend  
- **React** – Thư viện xây dựng giao diện người dùng  
- **Node.js** – Môi trường chạy JavaScript phía server  

Toàn bộ mã nguồn được chia thành nhiều bài học, thể hiện quá trình phát triển từng bước trong thư mục `MERN/src`.

## Cấu trúc mã nguồn  
Thư mục chứa toàn bộ mã nguồn minh họa và ví dụ thực hành:
```
MERN/src
```
Mỗi bài học tương ứng với một bản cập nhật mã nguồn cụ thể để tiện theo dõi tiến trình phát triển.
