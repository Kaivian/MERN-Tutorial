## ğŸ“¦ Getting Started

### Prerequisites

- **Node.js** (Latest LTS recommended)
- Package manager: npm, yarn, pnpm, or bun

---

### Installation

```bash
git clone <repository-url>
cd my-app
```

Install dependencies:

```bash
npm install
# or
yarn install
# or
pnpm install
# or
bun install
```

Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open http://localhost:3000 in your browser.
